# Quantz Financial Services Namibia - Website

## Overview
A landing page website for Quantz, an insurance brokerage firm in Namibia. This is a static marketing site built with React and Express.

## Architecture
- **Frontend**: React with TypeScript, Tailwind CSS, shadcn/ui components
- **Backend**: Express.js (minimal - serves static frontend)
- **Routing**: wouter for client-side routing

## Pages
- `/` - Home (landing page with hero, services, why choose us, how it works, testimonial, CTA, footer)

## Key Files
- `client/src/pages/home.tsx` - Main landing page with all sections
- `client/src/App.tsx` - App router
- `client/src/index.css` - Theme variables (teal/dark navy color scheme)
- `tailwind.config.ts` - Tailwind configuration

## Design
- Color scheme: Dark teal/navy (#0a2e3a) with teal accents
- Font: Poppins
- Sections: Navbar, Hero, Insurance Solutions, Why Choose Us, How It Works, Testimonial, CTA, Footer
